About This Application

This decentralized application allows you to store files on the Sepolia testnet with automatic storage optimization. Small files (<10KB) are stored directly on the blockchain, while larger files are stored on IPFS with only the hash recorded on-chain.

Getting Started: Make sure you have Sepolia testnet ETH in your MetaMask wallet. You can get free Sepolia ETH from sepoliafaucet.com
